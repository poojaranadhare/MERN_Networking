CSS:
----------
cascading style sheet
we can use css to design the html page.....
syntax :
selector : declaration
h1       :{color:red; font-size:20px}
p       :{color:red; font-size:20px}



How to add css into html document
---------------
3 ways to add:
1)inline  -->style is applied to particular elment 
2)internal -->inside the head tag
3)external  --> outside the head tag
