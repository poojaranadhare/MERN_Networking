Tag:
-----------
    which is opening and closing
    eg.
       <h1></h1>
       <body></body>
       <p></p>
    Type of tags:
    ---------
      1)paired tag -------> <h1></h1>
      2) unpaired tag  -----><hr>
      3)self closing tags -----><img src""/>
element:
------------
       <h1>This is H1</h1>
       which has opening and closing and in between it has content.
    type of element:
    ----------------
       1)inline element --->it will take required width
         if there is a space it will start with same line.
         eg:
         -------
         img,a,span
       2)block element----->it will start with new line 
        and takke the complete width.
        eg:
        ----------
        h1,h2,p,div
